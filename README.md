 Pet Store Automation Testing using Selenium WebDriver

 Project Overview:

This project is a mini automation testing application developed using **Java** and **Selenium WebDriver**. The automation script performs an end-to-end shopping flow on the JPetStore demo website by simulating user interactions such as product selection, cart updates, and checkout navigation.

 Objective:

The main objective of this project is to demonstrate the implementation of Selenium automation concepts, including:

* Browser initialization using WebDriver
* Locating web elements using XPath
* Performing click and text input operations
* Taking screenshots at different stages of execution
* Automating an e-commerce purchase workflow

 Features:

* Launches the JPetStore website using Microsoft Edge browser.
* Selects a product category (Cats).
* Opens a specific product from the catalog.
* Adds an item to the shopping cart.
* Updates the quantity of the selected item.
* Proceeds to the checkout page.
* Captures screenshots after every major action for reporting and verification purposes.

 Technologies Used:

* Java
* Selenium WebDriver
* Microsoft Edge Driver
* Eclipse IDE

 Workflow:

1. Open the JPetStore application.
2. Capture the home page screenshot.
3. Select the Cats category.
4. Open a product from the product list.
5. Add the selected item to the cart.
6. Update the item quantity.
7. Update the cart.
8. Proceed to checkout.
9. Capture screenshots throughout the process.

 Learning Outcomes:

This project helped in understanding:

* Selenium WebDriver architecture
* XPath locators and element identification
* Method creation and code reusability
* Screenshot handling using `TakesScreenshot`
* Basic automation framework design and test flow implementation

 Website Used:

https://petstore.octoperf.com/actions/Catalog.action

This project serves as a beginner-friendly example of automating an e-commerce application using Selenium WebDriver and demonstrates fundamental web automation techniques in Java.
